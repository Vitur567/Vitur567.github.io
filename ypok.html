В терминале сначала инициализируем git пишем команду git init в результате в корневой папке проекта получаем папку .git затем прописываем конфигурацию командой git config --local user.name "Evgeny" git config --local user.email eugen.pospelov@yandex.ru в папке .git появляется файл config затем набираем git status и появляется: Untracked files: (use "git add ..." to include in what will be committed) index.html src/ ypok.html nothing added to commit but untracked files present (use "git add" to track) затем набираем git add -A потом: git status создаем комит: git commit -a -m"first commit" Чтобы увидеть какие коммиты были и когда набрать команду: git log
